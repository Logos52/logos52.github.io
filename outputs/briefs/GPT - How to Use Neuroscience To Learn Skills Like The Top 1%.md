# GPT - How to Use Neuroscience To Learn Skills Like The Top 1%

## Core Thesis

The source emphasizes that improving a skill often requires unlearning old patterns, not merely adding new ones. Existing habits compete with new behaviors. Skill learning therefore requires attention to interference, feedback, and deliberate replacement.

## Compressed Takeaways

1. **Old habits resist replacement.** Existing patterns are neurologically efficient.
2. **Unlearning is part of learning.** The learner must weaken or bypass the old response.
3. **Attention controls rewiring.** Practice needs focused awareness of the target change.
4. **Feedback must be specific.** The learner needs to know what to change.
5. **Practice quality matters.** Repeating the old pattern strengthens the wrong circuit.
6. **Small corrections compound.** Skill improvement comes from repeated targeted adjustments.

## Practical Workflow

1. Identify the target behavior.
2. Identify the current automatic pattern.
3. Define the replacement pattern.
4. Practice slowly enough to notice the old habit.
5. Use feedback to correct drift.
6. Increase speed only after the new pattern stabilizes.
7. Revisit under varied conditions.

This fits with [[wiki/Techniques/Marginal Gains|Marginal Gains]] and [[wiki/Techniques/Kolbs Experiential Cycle|Kolbs Experiential Cycle]].

## What This Should Feel Like

It should feel like interrupting autopilot. The learner should notice the old behavior trying to run.

It should not feel like mindless repetition or accumulating theory about the skill.
