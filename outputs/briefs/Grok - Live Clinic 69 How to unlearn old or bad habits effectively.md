
**OPERATING MODEL: UNLEARN TO ACCELERATE SKILL ACQUISITION**

**Core Mechanism**  
Old habits run on autopilot (low energy, automatic). New skills require the high-energy thinking system to build and stabilize a precise mental model (high energy, effortful). The bottleneck is not building the new model—it is the old habit firing first and exhausting the high-energy thinking system before the new model locks in.  

Unlearning = isolate the exact cue → old response loop, then surgically replace the response by drilling three isolated stages: awareness of cue, judgment (pre-programmed decision), execution. Mental simulation of judgment first builds the new model with minimal fatigue. Repetition drives the new response into autopilot. Habits change only at the micro cue-response level, not the macro skill level.

**Causal Chain**  
1. Internal cue (overwhelm, boredom, messiness, risk of error) fires.  
2. Old response launches instantly and cheaply (rote, AI offload, avoidance, self-punishment).  
3. High-energy thinking system must both suppress the old response and build new model simultaneously → rapid fatigue.  
4. After 1-2 hours, old habit dominates despite intention.  
5. Long sessions produce exhaustion with zero model progress.  
6. Repeated failure creates belief "I can't change" and wasted months/years.

**Failure Modes**  
- Long grinding sessions (high-energy thinking system quits, old habit wins).  
- Adding more techniques without removing the interfering response.  
- Practicing the whole skill instead of the single constraining micro-response.  
- Replacing response while leaving the reinforcing feedback (dopamine, ease, emotional safety) intact.  
- Relying on awareness alone—no judgment scripts, no isolated drilling.  
- Treating emerging emotional blocks as noise instead of new cues to map.  
- Copying expert methods without first clearing your specific old response.

**Practical Feel**  
Think code debugging, not training montage. Find the exact trigger line (cue), write the patch (script), test in simulation, then run in live conditions. Short, targeted 5-15 minute drills beat 4-hour marathons. The moment you feel the old pull, you already know the exact replacement move—no deliberation required.

**System Implications**  
- Maintain a living "cue-response map" for every high-leverage skill/behavior.  
- For each entry, create one crisp script: "When [exact cue], I do [specific new response] instead."  
- Drill scripts mentally until they feel obvious and automatic.  
- Practice execution only on low-stakes triggers of that cue.  
- Identify the single constraining micro-step and drill only that.  
- To kill a bad habit: remove cue or cut its feedback loop, then install replacement response.  
- Measure progress by reduction in mental fatigue and speed of unlearning, not hours practiced.  
- Run this loop on every new skill before adding volume or complexity.  

Apply this and every skill acquisition curve steepens because the hidden tax of old habits disappears.
