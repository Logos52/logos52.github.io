## Schema Reference

| Property | Values |
| --- | --- |
| type | kolb |
| skill | skill being advanced |
| goal | optional goal being advanced |
| date | YYYY-MM-DD |
| status | Not started / In progress / Done |
| start-time | time started |
| finish-time | time finished |
| duration | length of reflection |
| previous-kolb | previous reflection in the chain |
| next-kolb | next reflection in the chain |

## Sorting Rule

Sort first by skill when using Kolb to improve an ability. Sort by goal when the reflection is mainly about progress toward an outcome. When both apply, link both and let the Kolb appear in both places.
