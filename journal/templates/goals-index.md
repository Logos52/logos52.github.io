## Schema Reference

| Property | Values |
| --- | --- |
| type | goal |
| status | active / completed / paused / archived |
| priority | important + urgent / important + not urgent / not important + urgent / not important + not urgent |
| start-date | YYYY-MM-DD |
| end-date | YYYY-MM-DD |
| last-edited | YYYY-MM-DD |
| linked-skills | skills this goal pulls toward |
| general-direction | one-line broad area |
| medium-term-anchor | specific, dated, observable target |

## Anchoring Convention

Set each skill's anchored goal to the goal title. Use linked skills on the goal page as a manual override or supplemental list.
