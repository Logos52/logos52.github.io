## Schema Reference

| Property | Values |
| --- | --- |
| type | skill |
| category | language / wellness / productivity / knowledge-system / mindset / meta-skill |
| status | active / paused / archived |
| priority | important + urgent / important + not urgent / not important + urgent / not important + not urgent |
| anchored-goal | link or name of the goal this skill serves |
| competency | UI / CC-low / CC-high / UC |
| dimension-focus | mindset / self-regulation / self-management / deep-processing / retrieval |
| current-level | X/10 |
| target-level | X/10 |
| final-level-metrics | observable criteria for target level |
| daily-practice | true / false |
| last-kolb | most recent Kolb |
| next-kolb-due | YYYY-MM-DD |
| linked-goals | related goals |

## Note on Kolb Linking

Kolbs should carry a skill field so they can be sorted under the relevant skill. Goal links can be added when a Kolb cycle exists to advance a specific goal.
