# Practice Consistency

**Current Level:** 3/10  
**Final Level:** 7/10  
**Competency:** CI (Consciously Incompetent)

### Final Level Metrics

- Deliberate technique use (BHS or SIR) at least once per week for 8 consecutive weeks
- Kolbs reflection completed after each session — not skipped
- Weekly evaluation running consistently without being reset or skipped
- Technique use feels like a recurring behaviour rather than a special occasion

### What This Skill Is

Practice Consistency is not a technique — it is the substrate that makes every other Learning Systems skill work. A BHS session run once does nothing. SIR without recurrence degrades. Kolbs without the full loop is just note-taking.

The gap at CI is: the techniques are known but their use is not yet habitual. Each session still requires a deliberate decision to begin. The 30-minute exercises feel like interruptions rather than default behaviour.

### How to Develop This Skill

- Attach technique use to an existing anchor (e.g. a specific day or slot)
- Keep sessions short enough to start without resistance — quality over duration initially
- Use the weekly evaluation as the consistency check, not a separate tracking system
- After each session, immediately set the next one (Kolbs chaining logic)
- When a session is skipped, name the friction rather than just re-scheduling
