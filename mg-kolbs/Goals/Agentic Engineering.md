### Anchoring & Dissecting the Goal

**General direction:**  
Move from one-off agent use into reliable, agent-directed workflows.

**Medium term anchored goal:**  
By the end of 2026, have at least three reusable agent workflows running reliably — where an agent can pick up a task, execute it, and return a usable output without significant manual correction.

#### What mindsets and attributes are needed?

| Mindset / Attribute | Current | Final | Notes |
|---------------------|---------|-------|-------|
| Systems thinking | 6/10 | 9/10 | Designing for reuse, not just solving once |
| Precision in instruction | 5/10 | 8/10 | Writing prompts and specs an agent can act on |
| Willingness to iterate | 6/10 | 8/10 | Treating failures as diagnostic data |

#### What skills are needed?

| Skill | Current | Final | Notes |
|-------|---------|-------|-------|
| Prompt Design | 4/10 | 7/10 | Writing instructions an agent can follow reliably |
| Workflow Design | 3/10 | 7/10 | Structuring multi-step agent processes |
| Verification | 3/10 | 6/10 | Checking and correcting agent output systematically |

#### Processes connected to this goal

- Improve one prompt, workflow, or instruction per week
- Ask an agent to inspect one small system per week
- Convert one repeated action into a reusable process per month
- Tighten one repo, note, or automation so an agent can use it better next time
