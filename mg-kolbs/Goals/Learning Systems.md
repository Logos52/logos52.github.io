### Anchoring & Dissecting the Goal

**General direction:**  
Make techniques produce the intended thinking — not correct-looking activity.

**Medium term anchored goal:**  
By the end of 2026, run deliberate learning sessions using BHS and SIR consistently enough that the techniques feel automatic and produce measurable schema growth.

#### What mindsets and attributes are needed?

| Mindset / Attribute | Current | Final | Notes |
|---------------------|---------|-------|-------|
| Deliberateness | 5/10 | 8/10 | Choosing constraints over shortcuts |
| Tolerance for ambiguity | 5/10 | 8/10 | Sitting with questions before reaching for answers |
| Reflection habit | 4/10 | 8/10 | Completing the cycle after every session |

#### What skills are needed?

| Skill | Current | Final | Notes |
|-------|---------|-------|-------|
| Practice Consistency | 3/10 | 7/10 | Making technique use recurrent, not occasional |
| BHS | 4/10 | 7/10 | Bear Hunter System — aimed, targeted retrieval |
| SIR | 5/10 | 7/10 | Spaced Interleaved Retrieval — maintenance and recall |

#### Processes connected to this goal

- Run one BHS session per week (Aim → Shoot → Skin)
- Run one SIR block per week
- Complete a Kolbs reflection after each meaningful session
- Identify one shortcut per week and choose a deliberate constraint instead
- Improve one learning-system note after each Kolbs cycle
