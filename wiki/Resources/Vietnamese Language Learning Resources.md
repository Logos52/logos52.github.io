# Vietnamese Language Learning Resources


The goal is to create a personal resource catalog, not to reproduce the course. Links and brief resource notes are preserved because the value is in the link list; the course flow is summarized into broad categories.

Resource count: 122 linked rows after de-duplication.

## Quick Use

- Start with beginner comprehensible input and sound/script resources.
- Choose a dialect deliberately when the source notes identify one.
- Add tooling only when it removes friction from immersion.
- Use native and harder-content lists for tolerance of ambiguity once basic orientation is stable.
- Track which resources are compelling enough to sustain daily exposure.
