# Mandarin Chinese Language Learning Resources


The goal is to create a personal resource catalog, not to reproduce the course. Links and brief resource notes are preserved because the value is in the link list; the course flow is summarized into broad categories.

Resource count: 137 linked rows after de-duplication.

## Quick Use

- Start with beginner comprehensible input, pinyin, and sound resources.
- Track simplified versus traditional resources when choosing immersion material.
- Add character-specific tools after pronunciation orientation is underway.
- Use harder channels, Netflix, and podcasts to build ambiguity tolerance once beginner input is stable.
- Keep pronunciation, character recognition, and immersion as separate but linked tracks.
