# Premortem Analysis

Premortem Analysis imagines that a plan has already failed, then works backward to identify the most likely causes.

## Core Idea

People are often too optimistic before execution. A premortem makes failure psychologically available before it becomes real.

## Method

1. State the plan.
2. Imagine it is months later and the plan failed.
3. Ask each participant to write possible causes privately.
4. Collect causes without debating them too early.
5. Cluster the causes.
6. Identify the highest-risk causes.
7. Modify the plan, create branch plans, or add checks.

## Why It Works

A premortem gives permission to voice concerns without sounding disloyal. It turns pessimism into service.

## Personal Use

Use it for:

- A new study system.
- A language-learning plan.
- A GitHub publishing workflow.
- A major course or certification.
- An AI-built software project.
