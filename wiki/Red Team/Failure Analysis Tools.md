# Failure Analysis Tools

Failure analysis tools help the Red Teamer find how a plan breaks before reality does the testing.

## Core Questions

- If this failed, what caused it?
- Which dependency is most fragile?
- Which assumption is most dangerous?
- What second- and third-order effects are likely?
- What would an adversary exploit?
- Which task creates the most downstream risk?
