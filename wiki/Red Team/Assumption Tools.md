# Assumption Tools

Assumption tools make hidden claims visible so they can be tested before they quietly control a decision.

## Core Questions

- What must be true for this plan to work?
- Which assumptions are necessary?
- Which assumptions are likely?
- Which assumptions are fragile?
- Which assumption would break the whole plan if false?
- Which assumption is actually a wish?

## Learning Translation

In studying, assumptions often hide as feelings:

- "I understand this."
- "This detail is not important."
- "I can retrieve this later."
- "This example is basically the same."
- "The grammar rule explains the usage."

Assumption tools convert those feelings into testable claims.
