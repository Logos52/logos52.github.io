# Refold Language Learning System

This page is the hub for the user's Refold-based language learning notes.

## Summary

The Refold course frames language learning as building an instinct through repeated, attentive contact with the language. The core move is not to memorize a language into existence. The core move is to make the language increasingly understandable, compelling, and frequent enough that acquisition can happen.

For this vault, Refold should connect with the user's existing learning system:

- [[wiki/Dimensions/Self-Management|Self-Management]] creates daily exposure.
- [[wiki/Dimensions/Self-Regulation|Self-Regulation]] chooses the right activity for the current bottleneck.
- [[wiki/Dimensions/Retrieval|Retrieval]] supports vocabulary, sound, and pattern reinforcement without replacing immersion.
- [[wiki/Techniques/Marginal Gains|Marginal Gains]] chooses the next small improvement.

## Core Operating Loop

1. Choose compelling content.
2. Pay active attention.
3. Use tools to make the content more comprehensible.
4. Notice words, sounds, characters, and grammar in real input.
5. Revisit useful material until it becomes easier.
6. Adjust the balance between preparation, interactive immersion, and freeflow.
