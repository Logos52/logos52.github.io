# Priority 0

Priority 0 is the long-term layer: the identities, skills, relationships, and directions that should keep receiving attention over time.

This page is a focus primer. Its job is to make the next meaningful action easier to start.

## Active Set

- Agentic Engineering
- Learning Systems
- Vietnamese
- Fitness
- Relationships

## Parked But Alive

- 中文

## Operating Model

```text
identity
-> emotional reason
-> current edge
-> 30-minute exercise
-> visible progress
-> next marginal gain
```

The page should create recurrence without becoming a tracker. It should reinforce identity without turning into self-talk. It should make progress visible without requiring exhaustive logs.

## Agentic Engineering

**Identity:** I build with agents.

**Emotional reason:** This is the new interface for making software, notes, tools, and systems.

**Current edge:** Move from one-off agent use into reliable agent-directed workflows.

**30-minute exercises:**

- Improve one prompt, workflow, or instruction.
- Ask an agent to inspect one small system.
- Convert one repeated action into a reusable process.
- Tighten one repo, note, or automation so an agent can use it better next time.

**Visible progress:**

- one commit,
- one improved instruction,
- one reusable prompt,
- one output page,
- one workflow lesson.

## Learning Systems

**Identity:** I learn by building systems, not by consuming information.

**Emotional reason:** This is the system that makes every other skill easier to advance.

**Current edge:** Make techniques produce the intended thinking instead of correct-looking activity.

**30-minute exercises:**

- Run one small [[wiki/Techniques/Bear Hunter System|Bear Hunter System]] session.
- Do one [[wiki/Techniques/Spaced Interleaved Retrieval|Spaced Interleaved Retrieval]] block.
- Write one [[templates/Kolbs Template|Kolbs]] reflection after a meaningful attempt.
- Identify one shortcut and choose a constraint for next time.
- Improve one learning-system note so it becomes more usable.

**Visible progress:**

- one better question,
- one retrieval result,
- one clearer map,
- one Kolbs insight,
- one named shortcut,
- one marginal gain.

**Next marginal gain:**

-

## Vietnamese

**Identity:** I acquire Vietnamese through repeated contact with real input.

**Emotional reason:** Vietnamese connects language, culture, place, relationships, and daily life.

**Current edge:** Build recurrence through immersion, noticing, and comprehension rather than grammar perfection.

**30-minute exercises:**

- Watch or listen to Vietnamese input with full attention.
- Save a few useful phrases.
- Notice one sound, grammar pattern, or repeated expression.
- Rewatch one short clip until it feels more familiar.
- Use a comprehension tool only enough to keep attention moving.

**Visible progress:**

- one watched video,
- one phrase list,
- one noticed pattern,
- one short summary,
- one improved comprehension lever.

**Next marginal gain:**

-

## Fitness

**Identity:** I train my body as part of my operating system.

**Emotional reason:** Fitness supports energy, confidence, mood, and long-term capacity.

**Current edge:** Make training recurring and visible without overcomplicating it.

**30-minute exercises:**

- Strength session.
- Mobility session.
- Walk.
- Zone 2 cardio.
- Recovery session.
- Plan the next workout.

**Visible progress:**

- workout completed,
- weight or reps logged,
- time or distance logged,
- mobility improvement noticed,
- energy or recovery noted.

**Next marginal gain:**

-

## Relationships

**Identity:** I maintain connection deliberately.

**Emotional reason:** Relationships decay when they are left to chance.

**Current edge:** Convert vague care into simple contact, repair, planning, or appreciation.

**30-minute exercises:**

- Message someone.
- Plan a call or visit.
- Repair a small misunderstanding.
- Express appreciation.
- Think through one relationship that needs more attention.
- Schedule a future touchpoint.

**Visible progress:**

- one message sent,
- one call planned,
- one repair attempted,
- one appreciation expressed,
- one next touchpoint scheduled.
